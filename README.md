# Projet_MasterMind

Voici le projet d’algorithmique et de programmation structurée consistant à développer un MasterMind en C, réalisé par :
Mathis Vagner — Paulin Bouteiller — Julie Kervizic


Installation

Téléchargez et placez les fichiers suivants dans le même dossier :
- Makefile
- mastermind_principal.c
- mastermind_utilities.c
- mastermind_utilities.h
- README.md


Compilation et exécution

1. Ouvrir un terminal

(PowerShell, CMD, Terminal Linux/macOS…)

2. Se placer dans le dossier du projet

Utilisez la commande cd (Change Directory) pour atteindre le dossier contenant le projet :

exemple : "cd chemin/vers/le/dossier"

Pour vérifier que vous êtes dans le bon dossier, utilisez :
- macOS / Linux : ls
- Windows : dir

Vous devez voir apparaître les fichiers du projet, dont le Makefile.

3. Compiler le programme
tapez : make

4. Lancer le programme

tapez : ./programme

Sur Windows avec MinGW, l’exécutable peut s’appeler programme.exe.


Note pour les utilisateurs Windows

make peut ne pas être installé par défaut.
Pour l’utiliser, vous pouvez :
- installer MinGW
  ou
- utiliser WSL (Windows Subsystem for Linux).


Commandes utiles du Makefile
- make -> Compile le projet
- make clean -> Supprime les fichiers .o et l'executable
